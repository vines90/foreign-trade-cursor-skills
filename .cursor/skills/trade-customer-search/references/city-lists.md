# 国家 → Top 城市清单

> 用户给国家级目标市场时，自动展开为 Top 5-10 个采购力强的城市分别搜索。
> 入选标准：经济体量 / 港口物流 / 行业聚集 / 人口规模。

## 用法

- 国家级输入（如 "Germany"）→ 取该国 Top 5（默认）/ Top 10（深度模式）
- 城市级输入（如 "Berlin"）→ 直接用，不展开
- 大区级输入（如 "Western Europe"）→ 取每国 Top 1-2

- 城市排名按 GDP / 人口 / 商业活跃度综合排序，定期校准
